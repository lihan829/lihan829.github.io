# Resources

A collection of research tools, datasets, and code developed for computational phonology and linguistic analysis.

---

## Code

### BUFIA-AR
- Implementation of Bottom-Up Factor Inference over autosegmental representations  
- Focus: tonotactic learning and structured representations  
- [GitHub](https://github.com/lihan829/BUFIA_AR)

### ARextractor
- Tool for extracting autosegmental representations from input forms  
- Converts linear input into multi-tier structures  
- [GitHub](https://github.com/lihan829/ARextractor)

---

## Datasets / Corpora

### Hausa Tonotactic Dataset
- ~600+ lexical items with annotated tonal structures  
- Used for learning experiments (BUFIA vs. MaxEnt)  
- Format: AR matrices + tonal strings  
- [Download](https://github.com/lihan829/BUFIA_AR/blob/main/data/hausa.txt)
